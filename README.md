# Retail Sales Performance Analysis

## Overview
This project analyses large-scale retail transaction data to identify key drivers of revenue, performance trends, and operational inefficiencies.

## Key Insights
- Revenue is highly concentrated in a single market (China)
- A small number of categories drive the majority of sales
- Discounting reduces average order value without increasing volume
- Customer behaviour strongly favours digital payments

## Business Impact
The analysis highlights structural risks (market concentration) and inefficiencies (discounting strategy), while identifying clear opportunities for optimisation.
